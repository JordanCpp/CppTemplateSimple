# Настройка для Windows (Добавить скриншоты)
1. Скачайте Visual Studio 2022 Community Edition.
2. После установки запустите студию.
3. Выберите из меню программы клонировать репозиторий
4. Скопируйте ссылку на данный гит
5. Нажмите кнопку клонировать
6. Нажмите на кнопку F5 (Запустить собранную программу)
